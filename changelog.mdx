## Upgrade Process

Follow these steps to upgrade your BeeFood integration to the latest version.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

Review the [release notes](#recent-releases) for breaking changes.

  </Step>
  <Step title="Update SDK" icon="download">

<CodeGroup tabs="npm,yarn">
````bash
npm install @beefood/sdk@latest
````
````bash
yarn add @beefood/sdk@latest
````
</CodeGroup>

  </Step>
  <Step title="Test Endpoints" icon="zap">

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.example.com/v1/orders
```

Verify responses match the updated schema.

  </Step>
  <Step title="Deploy" icon="rocket">

Push changes and monitor for issues.

  </Step>
</Steps>

<Callout kind="tip">
After upgrading, clear your cache and test in staging first.
</Callout>
