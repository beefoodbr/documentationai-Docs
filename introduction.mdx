## Overview

BeeFood provides a centralized space for your project documentation. You create organized docs, collaborate with your team, and maintain knowledge bases effortlessly. It supports Markdown, MDX components, and custom workflows to streamline your documentation process.

With BeeFood, you avoid scattered files and version conflicts. Teams access up-to-date information quickly, improving productivity across projects.

## Key Benefits

BeeFood delivers value for teams and individual contributors alike:

- **Centralized Management**: Store all docs in one searchable space.
- **Collaboration Tools**: Real-time editing and version history keep everyone aligned.
- **Customizable Workflows**: Embed code examples, diagrams, and interactive components.
- **Scalable for Projects**: Handle small guides or enterprise knowledge bases.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">

    Sign up at `https://app.beefood.com/register`.

  </Step>
  <Step title="New Space" icon="plus">

    Click **New Space** and name your project, like "MyApp Docs".

  </Step>
  <Step title="Add First Doc" icon="file-plus">

    Create `introduction.mdx` and start writing.

    <CodeGroup tabs="Markdown,MDX">
      ```markdown
      # Welcome
      This is your first doc.
      ```
      ```mdx
      ## Overview
      BeeFood makes docs easy.

      <Callout kind="tip">Try components!</Callout>
      ```
    </CodeGroup>

  </Step>
  <Step title="Publish" icon="globe">

    Preview and publish. Share the link with your team.

  </Step>
</Steps>
