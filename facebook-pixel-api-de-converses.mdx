Olá,

É possível rastrear os eventos do seu cardápio digital utilizando Facebook Pixel com API de Conversões. Ao configurar com sucesso, os seguintes eventos serão rastreados:

- “Visualizar conteúdo”, toda vez que um produto ou página é vista;

- “Adicionar ao Carrinho”, toda vez que um produto é adicionado carrinho;

- “Inicialização de Compra”, toda vez que preenche dados de login e inicia o finalização da compra na etapa de selecionar o endereço;

- “Compra”, toda vez que uma compra é realizada;

Parar criar seu Pixel e Configurar no seu Sistema BeeFood, siga os passos a seguir:

## Resumo

- [Acesse seu Gerenciador de Negócios do Facebook](#acesse-sue-gerenciador-de-negócios-do-facebook)

# Acesse sue Gerenciador de Negócios do Facebook

[**https://business.facebook.com/home/accounts**](https://business.facebook.com/home/accounts)

Agora na sua conta Business selecionada, acesse o menu superior e depois **“Gerenciador de Eventos”**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/1-1.png" width="1900" height="842" alt="" />

Em seguida selecione **“Conectar fontes de dados”** para iniciar a criação do seu pixel.

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/2.png" width="1900" height="842" alt="" />

Agora selecione a Conectar uma nova fonte de dados **Web** e depois em **Avançar**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel1.png" width="1900" height="842" alt="" />

Agora adiciona um nome para identificar seu Pixel, sugerimos o nome **“Cardápio Digital BeeFood”** para fácil identificação futura e depois clique em **Avançar**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel2.png" width="1900" height="842" alt="" />

Agora escolha como quer conectar o seu site selecionando **API de Conversões e Pixel da Meta** e depois **Avançar**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel3.png" width="1900" height="842" alt="" />

Para finalizar a criação agora clique em **Concluir.**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel4.png" width="1900" height="842" alt="" />

Agora é necessário capturar os dados de **Identificador** e **Token** para preencher no seu Sistema BeeFood.

Então para iniciar **selecione sua fonte de dados**, depois vá na **aba Configurações** e clique sobre **Identificação do conjunto de dados** para copiar.

Depois cole nas configurações do seu sistema:

**Cardápio Digital → Aba Marketing → API de Conversões e Pixel da Meta → Pixel**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel10-1.png" width="1900" height="842" alt="" />

Agora é necessário gerar o seu token de acesso.

Acesse novamente sua **Fonte de dados**, vá na **aba Configurações** e depois em **Configurar integração direta**, clique em **Gerar token de acesso**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel7-1.png" width="1900" height="842" alt="" />

Agora com o token gerado, clique sobre ele para copiar e cole no sistema:

**Cardápio Digital → Aba Marketing → API de Conversões e Pixel da Meta → Token**

<Image src="https://versao.beefood-r2.work/imagens-manual/facebook-pixel/pixel9-2.png" width="1900" height="842" alt="" />

Agora clique em **Salvar (F2)** para finalizar.

- Adição ao carrinho

- Compra finalizada
